# York Christmas Trip - GitHub Pages

A minimalist weekly travel planner built for static hosting.

## Features

- **Modern Design**: Full-bleed color blocks for each day
- **Structured Categories**: Destinations, Activities, Transport, Food & Lodging
- **Local Storage**: All data persists in browser localStorage
- **Zero Backend**: Pure HTML/CSS/JS, ready for GitHub Pages

## Deployment

1. Upload `index.html` to your GitHub repository
2. Enable GitHub Pages in repository settings
3. Select the branch containing `index.html`
4. Done! Your site will be live at `https://[username].github.io/[repo-name]/`

## Local Development

Simply open `index.html` in a browser. No server required.

## Data Storage

All schedule data is stored in browser localStorage under the key `york_christmas_schedule`. 
To reset, clear browser localStorage or use browser dev tools.
